# face_multi_task
* 人脸多属性任务

## 项目介绍
* 人脸多任务,包括脸部 98 关键点，性别（男/女），年龄。

## 项目配置
* 作者开发环境：
* Python 3.7
* PyTorch >= 1.5.1

## 数据集

* wiki_crop_face，并争对该项目进行制作，添加了人脸关键点标注信息。
